## How to Use

### Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/bedtime-tracker.git
cd bedtime-tracker
```

2. Open `index.html` in a web browser, or host it on GitHub Pages

### Using the App

1. **Start Bedtime** - Press the "Start Bedtime" button to begin tracking
2. **Complete Tasks** - Tap each cell when a child completes that task
3. **Earn Points** - The fastest kid for each task earns 1 point
4. **View Results** - See summary when all tasks are complete
5. **Spend Points** - Kids can redeem points for prizes

### Edit Mode

Click the edit icon to:
- Add or remove kids
- Add or remove tasks
- Rename kids or tasks
